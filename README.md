
ASM-LOCK-PICK
=============

    Execute your assembly code as is, defined in a *.asm. Pass in a predefined
    stack and watch the code execute.
    

    + Runs on Python version >= 2.6
    + Currently provides support for executors of:
        * 8 bit.
        * 16 bit.

    + Running the code:
        ./run.py <file>  <stack>
